Business Card Configuration Info


How Does It Work?

This card is designed to be compatible with the QMK firmware! As such, this is what will be used to customize the card's functionality. QMK is a firmware centered around mechanical keyboards and built with customization in mind, so it has a very wide range of capabilities that you can harness to make this thing do nearly anything you want it to.


Editing the Keymap with Vial

If all you want to do is edit the keymap and set up macros, you can use Vial!
Vial can be used either directly in a browser (as long as it supports WebUSB, sorry Firefox enjoyers), or as a downloadable desktop app.
Here are the basic steps to getting Vial up and running, adapted from their official first-time use guide:

> Click to view the list!
  • Step 1: Navigate to get.vial.today and choose your preferred way to start the application.
  • Step 2: Plug the business card into your device, if you haven't already.
  • Step 2.5: If running Vial in your browser, Click the "Start Vial" button. You should see a popup that lists any compatible input devices. Select the business card. If you downloaded Vial to your device, you don't have to do this.
  • Step 3: Start customizing! Simply select the key you would like to change, and assign it whichever key or function you would like. Any changes you make are saved automatically to the card.
